#  空投系统1.0

空投系统是BR类游戏中提供高级装备的重要机制，空投箱通常包含稀有武器、防具和其他战术道具，如三级头甲、吉利服等，本文将介绍如何快速配置和平精英的空投箱功能。
<br>

## 配置流程

1. 在GameMode中增加空投系统功能组件
2. 创建空投系统配置蓝图，并配置相关参数
3. 配置空投掉落相关蓝图和表格
4. 脚本中调用API以启用空投功能

### 配置空投数据

添加完成空投组件后，接下来需要配置具体的物资发放规则以及空投物资，在 ```GameMode``` 蓝图中搜素**Data Manager**属性，查看 ```GMData Source Global Settings``` 配置，如下所示。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/CIWMMimage.png)

具体需要配置的选项为```空投配置类```与```空投盒子数据```，其中，```空投配置类``` 主要负责空投的物资发放规则，如物资类别，掉落概率等，```空投盒子数据``` 主要配置可发放的全部空投物资。

> 系统默认使用和平精英原生的空投规则，可直接使用，如开发者希望重新进行配置，也可新建数据表与蓝图进行替换。

**空投配置类**

空投配置类配置具体的掉落规则，如掉落哪些类别的物资，以及每种物资掉落的概率。

首先，继承新建蓝图```BP Items Box Data Config```，并替换原空投配置类。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Uwbe3image.png)

打开蓝图，在 ```Air Drop Box Group Configs``` 处配置数据，如图所示：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/6Jtzlimage.png)

其中```Group Name```为物品组名，此处配置为*AirDrop*，```Config Items```下配置具体的物资类别以及对应的掉落概率，具体参数如下：

- Gategory Filter：物资类别名
- Probaility Percent：该类别掉落概率

如图所示，这里配置了三个类别的物资，分别为Weapon（武器），Lucky（吉利服），Armor（防具），其中武器和防具的掉落概率为100，即代表每次空投箱必掉落该类型物资，吉利服掉落概率为30，即代表每次只有30%的概率会掉落吉利服类物资。

**空投盒子数据**

空投盒子数据本质为一张表格，根据空投配置类配置的物资类别，加入相对应的具体的物资。

重新新建空投物资表，```右键 -> 各种各样 -> 数据表格```，选择```Items Box Data Table Row```。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/ERs9Vimage.png)

打开新建的表格，插入数据，如下所示：

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/lhfm8image.png)

这里配置了6行数据，可以看到，```Category```的配置和空投配置类中```Gategory Filter```的配置相对应，按照空投配置类中配置的规则，Armor类型的物资和Weapon类型的物资必掉落，此处配置了4条Weapon数据，掉落时将会按照权重从中选择一条物资。

每行数据具体配置参数如下：

- Category：物资类别，需与空投配置类中的Gategory Filter配置对应，否则无效
- Item Tag：物资标签
- Weight：掉落权重，若该类别物资被抽中掉落，则根据Weight值进行第二次随机，参考上述Weapon类别物资，范围为1- 10
- Items Data：具体的物资及数量

如上图所示，此处配置了一条Category为Armor的数据，Weight配置的为10，因为Armor必定掉落，而Armor类别的数据又只有这一条，所以此条数据下配置的物资一定会添加进空投箱中。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/R4gfqimage.png)

```Items Data```下，此处配置了两个物资数据，分别是三级头盔和三级护甲，数量皆为1，如此，空投触发后，这两个物资都会被添加进入空投箱中。

其他具体配置，开发者可下载附件工程进行查看。

---

### 启用空投

完成配置后，需要手动开启空投，以便飞机按照预定的路径和设置执行空投任务。

获取空投组件**aircraft**并执行 ``ExecuteFlightPlan`` 方法，当空投箱落地后即可验证掉落出的物资是否与空投配置相匹配。

```lua
		local gamemode = UGCGameSystem.GameMode;
    -- Get gamemode

    local aircraft = gamemode.TransportAircraft;
    -- Get aircraft component

    aircraft.DropLoc = Vector.New(21440.0,28050.0,150);
    -- Set Drop Location
    
    aircraft.InStringPath = UGCMapInfoLib.GetRootLongPackagePath() .. "Asset/Blueprint/Generator.Generator_C";
    -- Set Aircraft Generator

    aircraft:ExecuteFlightPlan();
```

- 添加两组元素，一组元素为一个信号圈。<br>
![企业微信截图_16868337088887.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868337088887.png)

- 时间参数<br>
【Delay Time】等待多少时间才出现蓝圈（以上飞机或者上一个信号圈结束为起点，和平现在只有第0个信号圈有值，其他信号圈都是0）。
【Safe Zone Appeartime】蓝圈驻留时间，即信号圈停止不动，等待缩圈的那段时间。
【Last Time】缩圈时间，即蓝圈从开始缩到与白圈重合的时间。
【Pain】信号圈伤害，影响游戏中人物的信号值和毒圈淘汰倒计时时长，伤害越高，信号值扣的越快，伤害越高，倒计时时长越短。当信号值达到0时，人物将会倒地或死亡。<br>
![企业微信截图_16868337169511.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868337169511.png)

- 信号圈参数<br>
【Use Custom Blue Point】第0个蓝圈才勾选，并且约定蓝圈半径，后续的蓝圈会以上一个白圈位置和半径为自己的位置和半径，无须勾选。
【Bluepoint】将BP_radiation的x,y值复制粘贴到这里，以此规定蓝圈中心点位置。
【Blueradius】蓝圈半径（单位：厘米）
【Use Custom White Point】如果勾上use custom whitepoint 以及输入了白圈中心坐标，如果合规的情况下（即白圈是蓝圈的内接圆，不存在交叉，在圆外的情况），那么蓝圈朝着指定白圈位置缩；如果不合规，则按不勾选走，并在ds日志中记录此次不合规原因。
【Alpha】越大则白圈和蓝圈越同心，取值范围是（0，蓝圈半径-白圈半径）<br>
![企业微信截图_16868337318637.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16868337318637.png)

```
local Action_TogglePoisonCircle = {}

function Action_TogglePoisonCircle:Execute(...)
        print("Action_TogglePoisonCircle:Execute " ..tostring(self));

        UGCBlueprintFunctionLibrary.TogglePoisonCircle(self, true);
        return false
end

return Action_TogglePoisonCircle
```

#  开启和平精英集成系统功能

在绿洲启元编辑器中，可以使用部分和平精英封装的集成系统功能，下面简单介绍如何在编辑器内开启这些功能。

## 操作步骤

1. 编辑器的GameMode蓝图，这里的GameMode使用的是UGCGameMode
