## 贴图的设置

贴图资源分为场景模型贴图、UI贴图、特效贴图、B通道法线贴图

### 1. 场景贴图设置

场景贴图分为固有色贴图、法线贴图、RMA贴图、

#### 1.固有色贴图设置：

![企业微信截图_16866449355526.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866449355526.png)

CompressionWithout Alpha：带有Alpha通道的贴图需要不勾选，勾选将去除Alpha通道。
Compression Settings:选择默认TC_Default。
Maximum Texture Size:贴图大小设置，根据实际情况。
SRGB：默认勾选。

#### 2.法线贴图设置：

![企业微信截图_16866449496890.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866449496890.png)

Compression Settings：法线贴图的压缩方式选择TC_Normalmap。
Texture Group :选择Normalmap 。
Not Used in Low Quality:勾选后低配不显示法线贴图。

#### 3.RMA贴图设置：
RMA：roughness(粗糙度)，metal(金属度)，ao(环境遮蔽)贴图，带这个贴图金属类物件整体表现会比较好。

![企业微信截图_16866449691944.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866449691944.png)

sRGB：不勾选

#### 4.B通道法线贴图
B通道法线贴图：把粗糙度放在了法线贴图的B通道，没有金属度贴图，对金属类物件表现不是很好。

![企业微信截图_16866449813524.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866449813524.png)

## 通用材质内容介绍

编辑器内置了一些具象材质和通用材质

### 场景通用材质介绍

![企业微信截图_16866450983253.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866450983253.png)

- 默认材质：默认材质用于表现一般物件，主要是金属物件，有RMA贴图（包括金属度），性能消耗比较大。
- 默认材质2：默认材质2用于比较大的室内建筑，需要单独烘焙的Lightmap贴图提高室内光影效果（builingAO）。
- 默认材质3: 默认材质3用于大部分物件，没有金属度，性能消耗比较小。

### 特效通用材质介绍

![企业微信截图_16866451153515.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866451153515.png)

- 特效材质1：所有像素值叠加不暗化，黑色渲染是透明的。例如火焰、灯光类效果。
- 特效材质2：半透明，黑色透明，白色完全不透明。例如烟。
- 特效材质3:特效材质1的基础上增加材质纹理的动态效果。
- 特效材质4:特效材质2的基础上增加材质纹理的动态效果。

### 3. 裁剪设置

![企业微信截图_16866421982344.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/%E4%BC%81%E4%B8%9A%E5%BE%AE%E4%BF%A1%E6%88%AA%E5%9B%BE_16866421982344.png)

- Culling Screen Size：模型裁剪尺寸设置，物体屏幕尺寸小于该数值模型将不显示。这个根据模型实际大小调整。
- Screen Size Culling Rough Distance：屏幕占比裁剪对应的裁剪距离
- Is Streamable：是否动态载入模型，蓝图类不合并模型需要勾选，mesh不勾选

<br>

## 动画序列

> 文档路径: 新手入门 > 资源管理与编辑 > 资源编辑 > 动画序列

#  动画序列

本章节介绍如何在绿洲启元编辑器中创建和使用过场动画序列（Sequence）。

<br>

## 基础概念

序列通常指过场动画序列。你可以将一系列元素，如Actor、组件，或是抽象的“事件”、“相机剪切” 等按时序规定他们的状态变化，组成一个“序列”。

例如，在关卡序列中规定某一相机随着时间变化，来调度镜头的轨迹，或是在Actor序列中规定某网格体组件的某材质，随着时间改变参数（如颜色）等等。绿洲启元编辑器下的序列有两种类型，第一种类型是关卡序列（LevelSequence），它用于规定某一关卡中的元素随时间的状态变化，第二种类型是Actor序列（ActorSequence），它用于规定某一蓝图下的元素随时间的状态变化。

<br>

## 关卡序列制作基础

### 1. 创建关卡序列并打开定序器（Sequencer）

在内容浏览器下新建一个`动画`-`关卡序列`的资源，双击新建的资源，打开关卡序列编辑器。

## 二、建筑材质

通用建筑材质适用于场景中的建筑类模型，如房屋、墙体、地面等。它支持完整的基础纹理、金属度、法线/粗糙度效果，以及可选的静态光照烘焙功能。

### 常用参数

| 参数 | 名称 | 作用 |
|------|------|------|
| `BaseColor` | 颜色贴图 | 模型的基础纹理，决定建筑表面的主要颜色和图案 |
| `MetallicEmissiveAO` | 金属度混合贴图 | 混合了模型的金属度、自发光和 AO 强度，一张图控制三种效果 |
| `NormalRoughness` | 法线粗糙度混合贴图 | 混合模型的法线（RG 通道）和粗糙度（G 通道），控制表面凹凸和光滑程度 |
| `NormalStrength` | 法线强度倍增器 | 控制法线贴图对表面凹凸感的影响。值为 1.0 保持原效果，大于 1.0 增强凹凸，小于 1.0 减弱 |
| `NormalStrength` | 法线强度倍增器 | 控制法线贴图对表面凹凸感的影响。值为 1.0 保持原效果，大于 1.0 增强凹凸，小于 1.0 减弱 |
| `RoughnessMultiplier` | 粗糙度倍增器 | 整体调整材质的粗糙程度，在 Roughness 贴图值基础上进行乘算 |

## 三、物件材质

通用物件材质适用于 UGC 环境中的各类模型，包括建筑和道具。相比建筑材质，物件材质更加轻量，同时**不支持光照贴图**。因此，物件材质更适合制作可交互物、小道具等不需要烘焙光照的模型。

### 材质变体

物件材质提供了三种变体，满足不同的渲染需求：`通用物件` / `通用物件（双面）` / `通用物件（Spline）`

#### 1. 通用物件

最常用的物件材质，适用于大多数模型。

| 参数 | 名称 | 作用 |
|------|------|------|
| `Albedo` | 基础贴图 | 模型的基础纹理 |
| `Normal` | 法线贴图 | 模型的法线贴图，增加表面细节感 |
| `RoughnessMetalicAoEmissive` | 混合贴图 | 混合了粗糙度、金属度、AO、自发光等信息 |
| `RoughnessMultiplier` | 粗糙度系数 | 控制模型粗糙度，取值越大表面越粗糙 |
| `Tint1` | 叠加颜色 | 在基础贴图上叠加颜色，改变模型的整体色调 |

#### 2. 通用物件（双面）

通用物件材质的变体，**支持双面渲染**。当模型需要从背面也能看到时（如布料、薄片、树叶装饰），可以选择此材质。

![image.png](https://cgugc-video-test-1258633575.cos.ap-shanghai.myqcloud.com/wiki_picture/Iciapimage.png)

#### 3. 通用物件（Spline）

通用物件材质的变体，**支持 SplineMesh 渲染**。适用于样条线生成的道路、围栏、管道等线性模型组件。

## 四、植被材质

植被材质专为植物类模型设计，支持基础颜色、次表面散射（透光效果）、风场动画和公告板渲染等多种效果，能显著提升植被的真实感。

### 材质变体

`植被材质` / `植被（顶点动画）` / `植被（公告板）`

#### 1. 植被材质（普通）

最常用的植被材质，适用于树叶、草丛等静态植被。支持次表面散射效果，模拟光线穿透叶片时的半透明感。

| 参数 | 名称 | 作用 |
|:----:|:----:|------|
| `Albedo` | 基础颜色贴图 | 植被的基础颜色纹理，包含树叶、草皮的固有色信息 |
| `BaseColorTint` | 基础颜色色调 | 整体偏移 Albedo 的颜色，相当于颜色滤镜。可以快速调整植被的整体色调 |
| `SSSIntensity` | 次表面散射强度 | 控制光线穿透植被表面的程度，模拟叶片透光效果 |
| `SSSSaturation` | 次表面散射饱和度 | 控制透光部分的颜色鲜艳程度，数值越高透光越鲜艳 |

##### 组合使用技巧

- **? Albedo + BaseColorTint：** 快速改变整片植被颜色——春天嫩绿（偏黄调）、夏天深绿、秋天黄绿、冬天枯黄，一套贴图即可适配四季变化。
- **☀️ SSSIntensity + SSSSaturation：** 模拟真实树叶透光效果——正面看叶子是深绿色，逆光时边缘呈现半透明的亮绿色，显著提升树木、草丛的真实感。

#### 2. 植被（顶点动画）

植被顶点动画材质用于实现**风吹摇摆效果**，让植被看起来更加生动自然。使用此材质的前提是模型本身带有顶点色权重，否则风场效果不会生效。

> ⚙️ **启用方式：** 将 `Wind` 参数开启，即可激活顶点动画效果。

| 参数 | 名称 | 作用 |
|:----:|:----:|------|
| `WindDistance` | 风影响距离 | 控制风的作用范围和最大偏移距离，限制顶点被吹动的最大位移量 |
| `WindIntensity` | 风强度 | 控制风吹的幅度，值越大摆动越剧烈 |
| `WindScale` | 频率缩放 | 控制摆动的随机一致性（0-1），越大吹动效果越杂乱 |
| `WindSpeed` | 风速度 | 控制风吹动画的时间频率，可以模拟风速快慢变化 |

##### 应用场景建议

| 场景 | 推荐设置 |
|------|----------|
| 树的根部 | 较小的 `WindDistance`（几乎不动） |
| 树梢 | 较大的 `WindDistance`（摆动幅度大） |
| 细长草叶 | 较大的 `WindScale`（摆动细碎） |
| 宽大叶片 | 较小的 `WindScale`（整体摆动） |
| 正常天气 | `WindSpeed` = 1.0 |
| 起风过程 | `WindSpeed` 从 0 逐渐增加到 1.5 |
| 暴风场景 | `WindSpeed` = 2.0 + 随机波动 |

#### 3. 植被（公告板）

植被公告板材质用于制作**远距离的植被效果**。使用公告板材质的模型会自动朝向摄像机，适合用于远景的树木、草丛等，以降低渲染开销。其参数含义与普通植被材质相同，额外支持面向相机的功能。

## 五、材质适用场景对比

三种材质各有侧重，以下是它们的核心差异对比，帮助您快速选择最合适的材质类型：

| 特性 | 建筑材质 | 物件材质 | 植被材质 |
|------|:--------:|:--------:|:--------:|
| **适用模型** | 房屋、墙体、地面 | 交互物、道具、建筑 | 树木、草丛、叶片 |
| **光照贴图** | ✅ 支持 | ❌ 不支持 | ❌ 不支持 |
| **金属度/粗糙度** | 独立混合贴图 | 单张混合贴图 | 不涉及 |
| **次表面散射** | ❌ | ❌ | ✅ 支持 |
| **风场动画** | ❌ | ❌ | ✅ 顶点动画 |
| **双面渲染** | ❌ | ✅ 有变体 | ✅ 公告板 |
| **SplineMesh** | ❌ | ✅ 有变体 | ❌ |

## 六、注意事项

> ⚠️ **物件材质不支持光照贴图：** 如果您需要烘焙静态光照的建筑模型，请务必使用**建筑材质**而非物件材质。物件材质适合用于交互物、小道具等不需要光照烘焙的场景。

> ⚠️ **顶点动画需要顶点色权重：** 使用植被顶点动画材质的模型，必须带有顶点色权重信息，否则风场效果不会生效。导入模型时请确认顶点色通道已正确设置。

### 选择材质的建议流程

1. **确定模型类型：** 是建筑、物件还是植被？
2. **检查是否需要光照烘焙：** 需要 → 建筑材质；不需要 → 继续判断
3. **检查是否有特殊需求：** 双面渲染 → 物件（双面）；Spline → 物件（Spline）；透光效果 → 植被材质；风吹动画 → 植被（顶点动画）
4. **创建材质实例：** 在对应通用材质的基础上创建实例，对贴图进行自定义赋值
